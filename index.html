49. This is a "default Nav Bar or Navbar Bootstrap component" which is mainly used for building website navigation menus. This "Navbar" is fully responsive and allows web developers to take advantage of it for various uses. Most web developers use a "Navbar" to either scroll a user all the way down the webpage to what they clicked on the "Navbar", or to take the user to another part of the website by making the "Navbar" items/elements have a link to the right of the hashtag < href="#" >, so when one of the "Navbar" items/elements is clicked the web browser automatically redirects itself to the other part of the website which is on another webpage. In addition, "Navbar" items can be active or inactive, which doesn't change the functionality of the item but rather its appearance. A good way to show a user that they are on the homepage, is to make the first "Navbar" item active, and the other ones inactive on that specific webpage.
